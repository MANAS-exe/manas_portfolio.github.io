Selected Projects
UG Major Project
SPATIALLY-AWARE ENSEMBLE LEARNING WITH RANDOM EFFECTS
Conventional regression breaks down when observations are spatially clustered — residuals become correlated, violating the independence assumption and biasing estimates.

Built a hybrid Random Forest – Random Effects (RF-RE) model that captures both nonlinearity (via Random Forest) and spatial correlation (via a GLS random-effects layer). The spatial variance-covariance matrix is estimated through Cholesky decomposition, and variance components are calibrated using GLS bootstrap. Feature selection uses VIF analysis to remove multicollinearity, followed by permutation importance from the RF stage.

Supervised by Dr. Arnab Hazra, IIT Kanpur.
R² = 0.88 — outperforms Linear Regression by +69%
Random Forest GLS Bootstrap Cholesky Decomposition VIF Analysis Spatial Statistics Python

STOCK PORTFOLIO MANAGER IN C++
A fully functional, OOP-based portfolio management engine in C++ built around real financial workflows — not a toy demo.

Maintains a persistent SQLite time-series store for historical price data fetched via Alpha Vantage API.
